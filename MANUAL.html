1. Visão Geral do Sistema

O Sistema de Operações – Coamo é uma plataforma web para acompanhamento e controle das rotinas diárias, semanais e mensais executadas pela equipe de TI. O sistema permite registrar o status de cada rotina, horários de início e fim, e sincroniza as informações em tempo real entre todos os usuários logados.

Acesso ao sistema:
🌐 https://opera-es-olive.vercel.app
Funciona em qualquer navegador (Chrome, Edge, Firefox) em qualquer computador ou celular.

Recursos principais

  • Controle de rotinas diárias, finais de semana e mensais
  • Acompanhamento do Corvu diário e mensal
  • Registro de horários de início e fim de cada rotina
  • Sincronização em tempo real entre usuários
  • Exportação para PDF das rotinas do dia
  • Gerenciamento de escalas por dia/mês na aba Horários
  • Controle de acesso por perfil (Operador / Admin)

4. Registrando Rotinas

Status das rotinas

Cada rotina possui um seletor de status na coluna direita:

  • ▶ Executando — Rotina em andamento. A linha fica destacada em azul.
  • ✔ Encerrado — Rotina concluída. A linha fica esmaecida após informar o horário final.
  • ✖ Erro — Rotina com problema. A linha fica vermelha e abre campo para descrever o erro e número do chamado.
Importante: A linha só fica esmaecida (cinza) quando o status é Encerrado E o horário de fim foi preenchido. Enquanto não tiver o horário final, a linha continua destacada mesmo com status Encerrado.

Registrando horários de início e fim

Ao marcar uma rotina como Executando, aparecem os campos de horário:

  • Campo Início — registre o horário que a rotina começou
  • Campo Fim — aparece quando o status é Encerrado ou Erro
Atalho de hora atual: Com o campo de hora em branco, pressione TAB para preencher automaticamente com a hora atual do computador.

Bloqueando uma rotina

O botão 🔒 ao lado de cada rotina serve para bloqueá-la quando não precisa ser executada naquele dia. A rotina fica com aparência riscada e não pode ser editada. Para desbloquear, clique em 🔓.

Rotinas com campos especiais

Algumas rotinas possuem campos de texto adicionais no lugar do seletor de status, como OK / Sem arquivo, Brasil / Itaú / Bradesco ou Servidor / Terminal. Preencha esses campos e saia do campo (Tab ou clique em outro lugar) para salvar automaticamente.

5. Corvu Diário e Mensal

As abas Corvu Diário e Corvu Mensal apresentam uma tabela com as rotinas do Corvu. Para cada linha, preencha:

CampoO que registrar
InícioHorário que a rotina do Corvu iniciou
FimHorário que a rotina finalizou
ServidorNome do servidor utilizado (ex: UNIFACEBG01)
UsuárioUsuário logado no Corvu (ex: OPERADOR1)
TerminalTerminal utilizado (ex: T01)
Salvamento automático: Os dados são salvos automaticamente ao sair de cada campo (Tab ou clique em outro lugar). O sistema sincroniza com o servidor e outros usuários veem as informações em tempo real.
⚠️ Corvu Mensal: As rotinas do Corvu Mensal devem ser executadas somente após atualizar a data mensal do IEPA no Uniface. O aviso aparece no topo da aba.

Bloqueando linha do Corvu

O botão 🔒 no final de cada linha permite bloquear uma rotina Corvu que não será executada. A linha fica riscada e os campos ficam desabilitados.

6. Sincronização em Tempo Real

O sistema sincroniza os dados entre todos os usuários a cada 3 segundos. Isso significa:

  • Se um operador marcar uma rotina como Executando, todos os outros usuários verão essa mudança em até 3 segundos
  • Não é necessário recarregar a página
  • O relógio no topo da tela atualiza a cada segundo
Indicador de salvamento: Quando há dados sendo enviados para o servidor, aparece o indicador ⏳ Salvando... ao lado do nome do usuário. Quando desaparece, os dados foram salvos com sucesso.

7. Gerenciamento de Horários

A aba 🗓️ Horários permite cadastrar as escalas de trabalho por dia. Apenas usuários com perfil Admin podem editar os horários.

Como adicionar um dia

1
Clique em + Adicionar Dia
2
Selecione a data no calendário
3
Clique em Adicionar
4
Preencha os turnos: horário, nome(s) e observação
5
Clique em + Turno para adicionar mais turnos no mesmo dia

Use as setas ‹ › para navegar entre os meses. O dia atual aparece destacado com o badge HOJE.

8. Exportar PDF

O botão 🖨️ Exportar PDF no topo da tela permite imprimir ou salvar como PDF a aba que está aberta no momento.

Como exportar

1
Navegue até a aba que deseja exportar (ex: Diário, Mensal, Corvu)
2
Clique em 🖨️ Exportar PDF
3
Na janela de impressão, selecione Salvar como PDF como destino
4
Clique em Salvar
O PDF gerado mostra apenas a aba ativa, com os status e horários registrados, sem os controles de edição. O nome do arquivo é gerado automaticamente com o tipo e a data (ex: Diaria_15-07-2026.pdf).

Comportamento da linha ao registrar status

  • Sem status — linha normal, fundo escuro
  • Executando — linha destacada com borda azul
  • Encerrado (sem horário final) — linha ainda destacada em azul
  • Encerrado (com horário final) — linha esmaecida (concluída)
  • Erro — linha com borda vermelha, campos para descrever o erro e número do chamado
  • Bloqueada — linha riscada com badge BLOQUEADA
